### Capstone group 111 

A low-cost, Raspberry Pi based remote-motion controller for a robotic claw.

Uses [pigpio](https://github.com/joan2937/pigpio/), the [Madgwick Fusion library](https://github.com/xioTechnologies/Fusion), and modified versions of [i2cdevlib](https://github.com/jrowberg/i2cdevlib), the [Adafruit QMC5883P driver](https://github.com/adafruit/Adafruit_QMC5883P), and the [Adafruit ADS1X15 driver](https://github.com/adafruit/Adafruit_ADS1X15).
